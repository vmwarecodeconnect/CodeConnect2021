    CODE 2771: Demystify Cloud Disaster Recovery to Build Resilient Environment

    October 6, 2021 | 11:00 AM PDT

    Disasters can come in any form like Natural disasters, Equipment and power outages, Cyberattacks and is a nightmare for any IT organization. The average cost of IT downtime is estimated to be close to $5,600 per minute. Disaster recovery as a service(DRaaS) is a cloud computing service model that allows an organization to back up its data and IT infrastructure in a third party cloud computing environment and provide all the DR orchestration, through a SaaS solution, to regain access and functionality to IT infrastructure after a disaster. In this session, we will discuss about a) DR planning b)key consideration for choosing the Disaster Recovery solutions c)solutions provided by key Cloud players in the market. The session will be concluded with a demonstration of DRaaS using VMware HoL.
